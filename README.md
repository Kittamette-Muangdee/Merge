# Merge
This repository is for Pre interviewing Form Intern 24 PRIMO

This project purpose is too merge 3 arrays into a single ascending sorted array base on these constraints:
-Collection_1 is sorted ascending.
-Colelction_2 is sorted ascending.
-Collection_3 is sorted descending.
-no sort functions are allowed to be used. 

# Instructions
<h2>1.Setup</h2>
You will need node.js and npm installed.
Create a folder with the given files inside

<h2>2.Install Dependencies</h2>
Open your terminal in root directory and type: <br>
npm install

<h2>3.Execute Unit Test</h2>
Type in your terminal: <br>
npm test <br>
Jest will automatically run the test case and show you results.
